# Portfolio Instructions

## Technology and Skill Icons

- Every new technology or skill badge must include its recognizable icon when one is available.
- Add the badge-to-icon mapping to `tagIcons` in `app/page.tsx` at the same time the badge is introduced.
- Prefer local icon assets in `public/assets/tech/` over remote URLs when practical.
- Do not leave a badge text-only without first checking whether an appropriate icon exists.
